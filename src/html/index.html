Basic Todo Application

API list

  1. api: /todo/

    method: GET

    description: To get list of all todos

  2. api: /todo/

    method: POST

    body: { 'title': 'TITLE_OF_TODO' }

    description: To add a new Todo item

  3. api: /todo/

    method: DELETE

    body: { '_id': 'ID_OF_THE_TODO_TO_DELETE' }

    description: To delete a single todo item
